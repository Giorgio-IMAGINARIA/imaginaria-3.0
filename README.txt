IMAGINARIA WEBAPP
A web application written in Angular 2 + ExpressJS + MongoDB

1) - git clone https://github.com/Giorgio-IMAGINARIA/imaginaria-3.0-gitHub.git
2) - type "npm install" in the root directory
3) - type "npm run build" in the root directory
4) - navigate to "http://localhost:3001/" in your browser
5) - enjoy :)
